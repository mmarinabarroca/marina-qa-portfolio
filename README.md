# QA Portfolio - Swag Labs

## Objective

This project demonstrates my skills in manual software testing, including test planning, test case design, test execution, defect reporting, and test evidence documentation.

## Application Under Test

## Scope

The following features were tested:

- User authentication (Login)
- Product listing and product details
- Shopping cart management
- Checkout process

## Testing Activities

The following activities were performed:

- Created a Test Plan
- Designed and executed Test Cases
- Documented Bug Reports

- Applied functional and exploratory testing techniques
