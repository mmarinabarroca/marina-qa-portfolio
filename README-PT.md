# Portfólio de QA – Swag Labs

## Objetivo

Este projeto demonstra minhas habilidades em testes manuais de software, incluindo planejamento de testes, criação de casos de teste, execução de cenários, documentação de defeitos e registro de evidências de testes.

## Aplicação Testada

Swag Labs (SauceDemo)

## Escopo

As seguintes funcionalidades foram testadas:

- Autenticação de usuário (Login)
- Listagem de produtos e detalhes dos produtos
- Gerenciamento do carrinho de compras
- Processo de checkout

## Atividades de Teste

As seguintes atividades foram realizadas:

- Criação de um Plano de Testes
- Elaboração e execução de Casos de Teste
- Documentação de Relatórios de Bugs
- Coleta de Evidências de Teste
- Aplicação de técnicas de testes funcionais e exploratórios

## Documentação de Testes

- Plano de Testes
- Casos de Teste
- Relatórios de Bugs
- Evidências de Teste

## Tipos de Teste

- Testes Funcionais
- Testes Exploratórios
- Testes Positivos
- Testes Negativos
